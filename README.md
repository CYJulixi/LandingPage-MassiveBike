# Massive Bike

Abre `index.html` en el navegador. Incluye catálogo, buscador, filtros, carrito persistente, servicios y formulario.

## GitHub Pages

1. Sube los tres archivos a la raíz de un repositorio.
2. Usa commits descriptivos para estructura, funcionalidad y diseño.
3. En **Settings > Pages**, publica la rama `main` desde `/root`.

## Guion de presentación

- Objetivo: convertir una página informativa en una tienda para ciclistas.
- Diseño: estilo deportivo, contraste y adaptación móvil.
- Código: HTML semántico, estilos separados y JavaScript para la interacción.
- Demostración: filtrar, buscar “casco”, agregar al carrito y recargar para comprobar que permanece guardado.
- Seguridad: el pago es demostrativo; una tienda real requiere un servidor y una pasarela de pago segura.

La solución apunta a los criterios de estructura, diseño, contenido, funcionalidad, calidad, creatividad, GitHub y presentación oral de la rúbrica.
